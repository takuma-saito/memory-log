D 言語入門
==========

D言語メモ
---------

* map! を使う時は *戻り値の型* を合わせなければならない
* map! の中で if などを使う時は *=>* を使わない
* partition を map で使う場合には事前に *array()* 関数を適用しなければならない
* array の popBack は副作用のある関数. *値は返さない* ので注意する
* 参照渡しの時は *ref* を付ける
* *文字列リテラルは immutable* なので気をつけること
* *double [][]* などの宣言は全てポインタであって、実体ではない。参照は別に実体を参照させる必要があるので注意すること！
* 無名関数は必ず戻り値を返すこと！！！
* D 言語の関数ポインタ *int function() fp;*, 関数を引数に渡す場合もewfこの形式で宣言すればよい
* ファイル名に `-` を入れてはならない
* foreach, range の [0..3] は範囲 0, 1, 2 を表す（終端の 3 は含まない）
* クオートで文字を囲むと、char 型になる。*string 型* ではない。
* *変換は cast ではなく to! で行うこと* なぜなら cast はアライメントを無視するため
* 配列の特定の番号に値を追加する時は *insert ではなく InsertInPlace を使う*
* 静的な配列は逆から読む
* sort! では **文字型はソートできない!!** その場合文字列型で sort! を行う
